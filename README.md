# Изучение C# и .NET

Этот репозиторий отражает мой путь от основ до сложных проектов.

## Структура

### Учебные упражнения
- `00-HelloWorld` — первые шаги
- `01-GuessGame` — условия, циклы, случайные числа
- `02-BankAccount` — ООП, классы, методы
- `03-GenericsDemo` — обобщения

### Проекты
Папка `Projects/` содержит более сложные, развиваемые приложения:
- `FinanceTracker` — личный финансовый трекер (консоль → веб → база данных)

## Технологии
- C# 10+
- .NET 8/9
- Git & GitHub
